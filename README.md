# oracle-automation-manager-OLAM-
preparing Execution environment for Oracle automation manager
Preparing and execution environment (EE) Container image for Oracle Automation Manager (OLAM) is difficult Taks as it fail many times due to dependencies.

In this document you will be able to find  how i have created the container images for OLAM to perform Ansible based Automation.
While Preparing i have focused based on Automation on windows Environment with Security.
This EE support SSL Encryption with kerberos Authetication for WINRM Connectity , which is secure.
